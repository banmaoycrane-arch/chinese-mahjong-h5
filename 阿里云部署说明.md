# 中国麻将 — 阿里云部署说明

与 `tower-defense-3d` **独立部署**，可共用同一台轻量服务器。

## 架构

```
玩家浏览器 ──→ http://公网IP:3001 ──→ PM2(chinese-mahjong) ──→ 静态 H5
塔防玩家   ──→ http://公网IP:3000 ──→ PM2(tower-defense)   ──→ 静态 H5 + 排行榜 API
```

## 服务器目录

```
/www/chinese-mahjong/
├── index.html
├── css/  js/
├── server/
│   ├── server.js
│   └── package.json
└── deploy/aliyun/
    ├── pm2.config.cjs
    └── nginx*.conf.example
```

## 首次部署

```bash
cd /www/chinese-mahjong/server
npm install --registry=https://registry.npmmirror.com
pm2 start ../deploy/aliyun/pm2.config.cjs
pm2 save
```

## 安全组

| 端口 | 用途 |
|------|------|
| 3001 | 麻将 H5（公网访问） |
| 22 | SSH |

## 有域名时

参考 `deploy/aliyun/nginx.conf.example`，配置 `mahjong.您的域名.com` 反代到 3001。

## 与塔防对比

| 项 | 塔防 | 麻将 |
|----|------|------|
| PM2 名 | tower-defense | chinese-mahjong |
| 端口 | 3000 | 3001 |
| 排行榜 | 有 | 暂无（纯单机） |
| 打包脚本 | 打包上传阿里云.bat | 打包麻将上传阿里云.bat |

详细亲友试玩步骤见 **`亲友试玩-无域名.md`**。
