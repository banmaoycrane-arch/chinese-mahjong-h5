# 亲友休闲试玩 — 中国麻将（无域名、阿里云 IP）

> 与塔防游戏**完全分开**：塔防用 3000 端口，麻将用 **3001** 端口。

---

## 一、和塔防的区别

| 项目 | 服务器目录 | 端口 | 试玩地址 |
|------|------------|------|----------|
| 3D 塔防 | `/www/tower-defense-3d` | 3000 | `http://IP:3000` |
| 中国麻将 | `/www/chinese-mahjong` | 3001 | `http://IP:3001` |

两个游戏互不影响，可同时在同一台阿里云上运行。

---

## 二、部署步骤（约 10 分钟）

### 第 1 步：本地打包

双击桌面：**`打包麻将上传阿里云.bat`**

生成文件夹：`chinese-mahjong-上传阿里云`

### 第 2 步：上传到服务器

用 WinSCP 或阿里云文件管理，上传到：

```
/www/chinese-mahjong/
```

### 第 3 步：服务器启动（SSH）

```bash
cd /www/chinese-mahjong/server
npm install --registry=https://registry.npmmirror.com
pm2 start ../deploy/aliyun/pm2.config.cjs
pm2 save
```

检查：

```bash
curl http://127.0.0.1:3001/api/health
```

应返回 `"service": "chinese-mahjong"`

### 第 4 步：阿里云防火墙

添加入站规则：

| 端口 | 协议 | 说明 |
|------|------|------|
| **3001** | TCP | 麻将游戏 |

（塔防 3000 若已放行，无需改动）

### 第 5 步：试玩

```
http://您的公网IP:3001
```

选难度 → 开始对局。

## 三、发给朋友


---

## 四、更新游戏

1. 本地重新运行 `打包麻将上传阿里云.bat`
2. 上传覆盖 `/www/chinese-mahjong`（不必删 `server/node_modules`）
3. 服务器执行：

```bash
pm2 restart chinese-mahjong
```

## 五、常用命令

```bash
pm2 list                      # 应看到 tower-defense + chinese-mahjong

pm2 restart chinese-mahjong   # 重启麻将
```

## 六、常见问题

**Q：和塔防会冲突吗？**  
不会。目录、端口、PM2 进程名都不同。

**Q：80 端口被占了怎么办？**  
直接用 `http://IP:3001` 即可，无需 Nginx。

**Q：想用 8080 不带长端口？**  
可参考 `deploy/aliyun/nginx-mahjong-ip.conf.example` 配置 Nginx 反代。
