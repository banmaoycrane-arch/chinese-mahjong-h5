# 中国麻将小游戏

独立 H5 项目，与桌面上的「塔防游戏」**完全隔离**，单独文件夹、单独启动，方便后续管理。

**GitHub 仓库（独立）：** https://github.com/banmaoycrane-arch/chinese-mahjong-h5

## 快速开始

**方式一（推荐）：** 双击桌面 `Play-Chinese-Mahjong.bat`

**方式二：** 进入 `chinese-mahjong` 文件夹，双击 `index.html` 用浏览器打开

**方式三：** 双击 `chinese-mahjong\启动麻将.bat`，访问 http://localhost:8081

> 请用 Chrome / Edge / Safari 等浏览器运行，不要在代码编辑器内预览。

## 游戏说明

### 基本规则（国标简化）

- **人数：** 4 人，你为东家（下方），其余三家为电脑
- **牌数：** 136 张（万、筒、条各 36 张，风牌 16 张，箭牌 12 张）
- **流程：** 开局两次掷色子（定牌墙 → 定起抓墩）→ 洗牌砌墙 → 发牌 → 摸牌 → 打牌 → 吃碰杠胡；杠牌从牌尾补牌并掷色子
- **吃：** 只能吃上家打出的牌
- **碰 / 杠：** 任意一家打出的牌均可
- **胡牌：**
  - 标准型：4 组面子（顺子或刻子）+ 1 对将
  - 特殊型：七对（7 个对子）
- **计番：** 自摸 +1 番，七对 +2 番等，难度越高输赢倍数越大

### 掷色子（定墙 / 定墩）

与各地麻将惯例一致，开局掷**两次**色子，决定从牌墙哪里起抓：

1. **第一次（定牌墙）：** 庄家掷两颗色子，点数和从庄家起**逆时针**数，落到谁家就从谁家面前的牌墙起数。
2. **第二次（定墩）：** 再掷两颗色子，从该道牌墙**右端**向内数墩，确定起抓的墩位。
3. **发牌：** 跳过起抓处一墩（休墩），再按逆时针每人 4 张 × 3 轮，最后每人再摸 1 张，共 13 张；庄家开局再摸一张。
4. **杠补牌：** 杠牌后掷色子，从**牌尾**（开牌处相反一端）补一张牌。

左侧面板会显示本局起抓位置；杠牌时会播报补牌色子点数。

### 座位（逆时针）

| 位置 | 座位 |
|------|------|
| 下方 | 东家（你） |
| 右侧 | 南家（电脑） |
| 上方 | 西家（电脑） |
| 左侧 | 北家（电脑） |

### 四档难度

| 难度 | 说明 |
|------|------|
| 🌱 简单 | 电脑常打错牌，很少吃碰杠，有操作提示 |
| ⚔️ 普通 | 均衡体验，适合新手熟悉规则 |
| 🔥 困难 | 电脑较积极，防守更好 |
| 💀 地狱 | 电脑几乎最优打法，赢面很低 |

难度选择会保存在浏览器本地，下次打开自动记住。

## 操作方式

### 电脑 / 笔记本

- 点击手牌选中，再点「打出」按钮
- 出现「吃 / 碰 / 杠 / 胡」时点击对应按钮，或点「过」
- 左上角 ☰ 返回选难度

### 平板 / 手机

- **推荐横屏**，视野更开阔
- 竖屏时会提示旋转，也可点「知道了」继续玩
- 触控选手牌、点按钮操作

## 与塔防项目的关系

| 项目 | 文件夹 | 本地启动 | 阿里云端口 |
|------|--------|----------|------------|
| 塔防游戏 | `tower-defense-3d` | `Play-Tower-Defense.bat` | 3000 |
| 中国麻将 | `chinese-mahjong` | `Play-Chinese-Mahjong.bat` | 3001 |

两个项目互不引用、不共用代码，可分别部署、分别维护。

## 部署到阿里云

1. 双击桌面 **`打包麻将上传阿里云.bat`**
2. 上传到服务器 `/www/chinese-mahjong`
3. SSH 执行 `npm install` + `pm2 start`
4. 防火墙放行 **3001** 端口
5. 访问 `http://公网IP:3001`

详细步骤见 **`亲友试玩-无域名.md`** 和 **`阿里云部署说明.md`**。
